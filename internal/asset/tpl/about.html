Proxy manager.

Support http、https、socks4、socks4a、socks5、shadowsocks proxy.

Project : https://github.com/hidu/proxy-manager

Current Version: {{.version}}



Usage:

1. As HTTP/HTTPS Proxy Server

PHP:
ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://hidu.github.io/hello.md"); 
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_PROXY,"{{.proxy_host}}:{{.proxy_port}}");
curl_setopt($ch, CURLOPT_PROXYUSERPWD,"$name:$psw"); // if necessary
$body=curl_exec($ch); 
curl_close($ch);

Go HTTP.Client:
proxyURL, err := url.Parse("http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}")
if err != nil {
    log.Fatalf("parser proxy %q: %v\n", *proxy, err)
}
client := &http.Client{
    Timeout: 10 * time.Second,
    Transport: &http.Transport{
        Proxy: func(req *http.Request) (*url.URL, error) {
            log.Println("using proxy", proxyURL.String())
            return proxyURL, nil
        },
    },
}

Bash CURL:
curl -x http://$name:$psw@{{.proxy_host}}:{{.proxy_port}} 'https://hidu.github.io/hello.md'

2. API

2.1 /query

Handle it as a normal web service and proxy the request.

curl 'http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/query? url=urlencode($url) & method=urlencode($method) & header=urlencode({"k":"v"})

e.g.:

curl 'http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/query?url=https://hidu.github.io/hello.md

 curl 'http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/query?method=POST&url=https://hidu.github.io/hello.md&header={"k1":"v1"}' \
  -X POST --data "request body"

2.2 /fetch

Get an available proxy.

 curl 'http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/fetch'
