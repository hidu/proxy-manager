Usage:

1. As HTTP/HTTPS Proxy Server

Bash CURL:
    curl -x http://$name:$psw@{{.proxy_host}}:{{.proxy_port}} 'https://hidu.github.io/hello.md'
    
PHP:
    ch = curl_init();
    curl_setopt($ch, CURLOPT_URL, "https://hidu.github.io/hello.md");
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_PROXY,"{{.proxy_host}}:{{.proxy_port}}");
    curl_setopt($ch, CURLOPT_PROXYUSERPWD,"$name:$psw"); // if necessary
    $body=curl_exec($ch);
    curl_close($ch);
    
Go HTTP.Client:
    proxyURL, err := url.Parse("http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}")
    if err != nil {
        log.Fatalf("parser proxy %q: %v\n", *proxy, err)
    }
    client := &http.Client{
        Timeout: 10 * time.Second,
        Transport: &http.Transport{
            Proxy: func(req *http.Request) (*url.URL, error) {
                log.Println("using proxy", proxyURL.String())
                return proxyURL, nil
            },
        },
    }

2. API

2.1 /fetch

Fetch the target URL via a configured proxy server.

http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/fetch? url=urlencode($url) &method=POST &header=urlencode({"k":"v"}) &retry=3 &format=clean &filter=urlencode(tag1&tag2,tag3,[ANY])

  • url: required, a valid URL starting with http:// or https://
  • method: optional, HTTP method (default: "GET")
  • header: optional, HTTP headers in JSON format (e.g. {"k":"v"})
  • retry: optional, number of retry attempts
  • format: optional, available value: clean — returns sanitized HTML (with JS, CSS, etc. removed)
  • filter: optional, filters proxies whose Tags field satisfies this condition (e.g. tag1&tag2,tag3,[ANY] )

e.g.:

curl 'http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/fetch?url=https://hidu.github.io/hello.md'
curl 'http://$name:$psw@{{.proxy_host}}:{{.proxy_port}}/fetch?method=POST&url=https://hidu.github.io/hello.md&header={"k1":"v1"}' \
      -X POST --data "request body"

2.2 /direct

Fetch the target URL using the local network without a proxy.

Parameters are the same as /fetch.

2.3 /pick
